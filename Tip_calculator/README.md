# Day 02 - Tip Calculator

This project is part of my **100 Days of Python Challenge**.

The Tip Calculator helps users calculate how much each person should pay when splitting a bill, including the tip percentage.

## Project Description

The program asks the user for:

* Total bill amount
* Tip percentage (10%, 12%, or 15%)
* Number of people splitting the bill

It then calculates the total bill including tip and divides it equally among the people.

## Features

* Accepts user input for bill amount
* Calculates tip based on selected percentage
* Splits the bill among multiple people
* Displays the amount each person needs to pay
* Formats the output to two decimal places

## Technologies Used

* Python
* Basic arithmetic operations
* User input handling
